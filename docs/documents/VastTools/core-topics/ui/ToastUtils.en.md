# Toast

## Long Toast

!!! note "Long toast"

    === "Kotlin"

        ```kotlin
        ToastUtils.showLongMsg(this, "Hello") // use characters directly
        ToastUtils.showLongMsg(this, R.string.app_name) // use str id
        ```

    === "Java"

        ```java
        ToastUtils.showLongMsg(this, "Hello") // use characters directly
        ToastUtils.showLongMsg(this, R.string.app_name) // use str id
        ```
